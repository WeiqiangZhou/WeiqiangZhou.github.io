Software & Tools

Open-source computational tools for genomics and single-cell analysis

R Package

SCATE

Single-Cell ATAC-seq signal Extraction

SCATE extracts and enhances signal from sparse single-cell ATAC-seq data. It leverages information across cells and genomic loci to impute regulatory activity with high accuracy, enabling downstream analysis of cell-type heterogeneity and regulatory dynamics.

💻 GitHub 📄 Paper
R / Command Line

BIRD

Big Data Regression for DNase-seq

BIRD predicts genome-wide DNase I hypersensitivity (chromatin accessibility) from gene expression data — both microarray and RNA-seq. Using a pre-built model, it generates predictions for ~1 million genomic loci and outputs data in matrix or WIG format for UCSC Genome Browser visualization.

💻 GitHub 📄 Paper (Nat. Comm.)
Web Database

PDDB

Predicted DNase I Hypersensitivity Database

PDDB is a comprehensive database of predicted DNase I hypersensitivity profiles for thousands of biological samples, generated from gene expression data using BIRD. It allows researchers to query and download chromatin accessibility predictions across diverse cell types and conditions.

🌐 Web App
R / Shiny App

SCRAT

Single-Cell Regulome Analysis Toolbox

SCRAT is a toolbox with a graphical user interface for studying cell heterogeneity using single-cell regulome data (scATAC-seq, scChIP-seq, etc.). It provides clustering, visualization, and regulatory feature analysis in an interactive Shiny interface — no coding required.

💻 GitHub 📄 Paper (Bioinformatics)
