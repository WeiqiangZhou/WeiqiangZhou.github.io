Software & Tools

Open-source computational tools for genomics and single-cell analysis

R Package

SCATE

Single-Cell ATAC-seq signal Extraction and Enhancement

SCATE leverages information across cells and genomic loci to impute and enhance signal in sparse single-cell ATAC-seq data. It enables accurate characterization of regulatory activity and cell-type heterogeneity from chromatin accessibility profiles.

💻 GitHub 📄 Paper (Genome Bio.)
R / Command Line

BIRD

Big Data Regression for DNase I Hypersensitivity

BIRD predicts genome-wide chromatin accessibility (DNase-seq signal) from gene expression data — both microarray and RNA-seq. Using a pre-built model it generates predictions across ~1 million genomic loci and outputs WIG or matrix format for UCSC Genome Browser visualization.

💻 GitHub 📄 Paper (Nat. Comm.)
Web Database

PDDB

Predicted DNase I Hypersensitivity Database

A comprehensive database of predicted DNase I hypersensitivity profiles for thousands of biological samples, generated using BIRD from gene expression data. Allows researchers to query and download chromatin accessibility predictions across diverse cell types and conditions.

🌐 Web App
R / Shiny

SCRAT

Single-Cell Regulome Analysis Toolbox

SCRAT is an interactive toolbox with a Shiny GUI for analyzing single-cell regulome data (scATAC-seq, scChIP-seq, etc.). It provides clustering, visualization, and regulatory feature analysis — no coding required.

💻 GitHub 📄 Paper (Bioinformatics)
